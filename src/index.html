File a guest with sanity,
not with the official site.

Croatian hosts must report every overnight guest to eVisitor. The official UI is the kind of thing you only open when you have to. This generates a tidy TouristCheckIns.xml you can upload once via Turisti → Prijava putem datoteke. No login, no analytics, nothing leaves your browser.

How it works
  1. 1 Save your Facility code and defaults — kept on this device.
  2. 2 Add one entry per guest. Duplicate a row for families.
  3. 3 Download the file, log in, upload, done.
